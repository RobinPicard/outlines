def adapt_tokenizer(tokenizer: PreTrainedTokenizerBase) -> PreTrainedTokenizerBase: ¶

Adapt a tokenizer to use to compile the FSM. The API of Outlines tokenizers is slightly different to that of `transformers`. In addition we need to handle the missing spaces to Llama's tokenizer to be able to compile FSMs for this model. Parameters ---------- tokenizer The tokenizer of the model. Returns ------- PreTrainedTokenizerBase The adapted tokenizer.
