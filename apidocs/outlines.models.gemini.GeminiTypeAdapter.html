class GeminiTypeAdapter(ModelTypeAdapter):

View In Hierarchy

Type adapter for the Gemini clients.

`GeminiTypeAdapter` is responsible for preparing the arguments to Gemini's `generate_content` methods: the input (prompt and possibly image), as well as the output type (only JSON).
