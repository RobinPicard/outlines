outlines.applications.Application

class Application:

Constructor: Application(template, output_type)

View In Hierarchy

Application is a class that encapsulates a prompt template and an output type. It can be called to generate a response by providing a model and optional arguments used to create the prompt from the template. Parameters ---------- template : Union[Template, Callable] A callable that takes arguments and returns a prompt string. output_type : Any The expected output type of the generated response. Examples -------- ```python from pydantic import BaseModel from transformers import AutoModelForCausalLM, AutoTokenizer from outlines import models, Application from outlines.types import JsonType from outlines.templates import Template class OutputModel(BaseModel): result: int model = models.from_transformers( AutoModelForCausalLM.from_pretrained("microsoft/Phi-3-mini-4k-instruct"), AutoTokenizer.from_pretrained("microsoft/Phi-3-mini-4k-instruct") ) template_string = "What is 2 times {{ num }}?" template = Template.from_string(template_string) application = Application(template, JsonType(OutputModel)) result = application(model, num=3) print(result) # Expected output: { "result" : 6 } ```
