class OpenAI(Model):

View In Hierarchy

Thin wrapper around the `openai.OpenAI` client.

This wrapper is used to convert the input and output types specified by the users at a higher level to arguments to the `openai.OpenAI` client.

def stream(self, *args, **kwargs): ¶
overrides outlines.models.base.Model.stream

Stream a response from the model. Users can use the `stream` method from the model directly, in which case we will create a generator instance with the output type provided and then invoke its `stream` method. Thus, those commands are equivalent: ```python generator = Generator(model, Foo) for chunk in generator("prompt"): print(chunk) ``` and ```python for chunk in model.stream("prompt", Foo): print(chunk) ```
