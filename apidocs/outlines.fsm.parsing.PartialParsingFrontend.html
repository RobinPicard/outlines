def _compute_termset_fsm_info(self): ¶

Collect and return information about terminal symbol sets and their FSMs.

Terminal symbol sets (or "termsets") are ordered sequences of terminal symbols that are used by each parser state. Associated with each is a collection of FSMs for each terminal and a single parse state FSM that is the union of each terminal's FSM.

This constructs a list of tuples containing the termset, the set of parse states that use the termsets, parse state FSMs, and information mapping the components of the parse state FSMs to their terminal symbol FSMs.
